Direct Assignment VS. Assignment By Reference

/* Assigning a primitive value to a variable, any changes to that variable is made to its value */
const a = 1;
let b = a; // a = 1, b = 1
b = 2; // a = 1, b = 2
/* But if you assign non-primitive value to a variable, then it is done by reference */
const c = { value: 1 };
let d = c; // c.value = 1, d.value = 1
d.value = 2; // c.value = 2, d.value = 2

Functions

There are many ways to declare functions in JS: Constructor funtion, function literal, arrow function, and the normal way to declare a function is:
function functionName() {
// function body
}

Declaring functions like the example above will make the function be hoisted automatically (you can call the funtion on any part of the program).

Callbacks
