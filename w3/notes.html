Objects

Arrays and funtions are actually objects (see console).

Object literal in JS is a self-contained set of related values (properties) and funtions (methods).

In JS, you don't have to define a class to declare an object literal

Calling Methods using square bracket notation:
superman['fly'](); //parenthesis needed outside the brackets. See the console

JS code can be use to be a property key: place them inside the square bracket
const hulk = { name: 'Hulk', ['catch' + 'Phrase']: 'Hulk Smash!' }; //(see console)

You can check if an object has a specific property and work with them:
for(const key in superman) { if (superman.hasOwnProperty(key)) { console.log(key + ": " + superman[key]); } } // to check for its own prperties and log them on the console not including inherited properties

JSON

parse() takes a string of data in JSON format and returns a JS object.

stringify() takes JS object and returns a string of JSON data.

Document Object Model

DOM represents an HTML document as a network of connected nodes that form like a tree structure (node tree).

A node is a basic data structure and it contains data that may link to other nodes.

The DOM treats everything on a web page as a node (e.g. HTML tags, content of the tags, attributes, etc.)
